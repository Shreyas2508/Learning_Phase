# 💰 LoanWise

LoanWise is a simple Python project that reads loan data from text files, processes it, and calculates monthly EMI (Equated Monthly Installment) values.

## 📌 Features
- Reads loan data from `.txt` files  
- Parses and structures data using Python  
- Calculates EMI using a mathematical formula  
- Writes results into new output files  
- Modular code with helper functions
